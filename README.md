# Amazon-Clone
# 🛒 Amazon UI Clone

A fully responsive Amazon homepage clone built using **HTML and CSS**, replicating core UI components like navbar, hero image slider, and product sections.

---

## 🚀 Features

* 🧭 Navigation bar similar to Amazon
* 🎞️ Hero image slider (carousel effect)
* 🛍️ Product sections with grid layout
* 📱 Responsive design for different screen sizes
* 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox & Grid)

---

## 📂 Project Structure

amazon-clone/
│── index.html
│── style.css
│── images/

## 🎯 Purpose

This project was created to improve frontend development skills by replicating a real-world e-commerce UI.
